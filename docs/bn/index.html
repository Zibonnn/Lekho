Homebrew আছে? একটাই কমান্ড:

অথবা GitHub Releases থেকে লেটেস্ট .dmg ফাইলটি নামিয়ে নিন; ওপেন করে Install Lekho.pkg-এ ডাবল-ক্লিক করুন — বাকিটা ইনস্টলার সামলাবে। macOS ইনস্টলার ব্লক করলে System Settings → Privacy & Security-এ যান, নিচে স্ক্রল করে লেখো-র পাশের "Allow Anyway" চাপুন।

ইনপুট সোর্স যোগ করুন

System Settings → Keyboard → Input Sources → Edit খুলুন, + বাটনে ক্লিক করুন, English → Lekho খুঁজে যোগ করে নিন। লিস্টে না দেখালে একবার লগ আউট করে লগ ইন করুন — নতুন কিবোর্ড চেনাতে macOS-এর মাঝে মাঝে এটুকু দরকার হয়।

জানতে চাওয়া কথা

এটা কি 'Avro Keyboard for Mac'-এর মতোই?

লেখো সেই একই ধারার উত্তরসূরি — Windows-এ Avro Keyboard আর macOS-এ iAvro যে অভ্র ফোনেটিক নিয়মগুলো জনপ্রিয় করেছিল, লেখো ঠিক সেই নিয়মেই চলে। iAvro এখনো কাজ করে, কিন্তু সেটা Intel-only — Apple Silicon-এ Rosetta দিয়ে চালাতে হয়, আর Apple ঘোষণা করেছে Rosetta ধীরে ধীরে বন্ধ হবে। লেখো শুরু থেকেই Apple Silicon-এর (M1, M2, M3, M4, M5) জন্য তৈরি — macOS যত এগোবে, লেখো সাবলীলভাবেই চলতে থাকবে।

M4 বা M5-এর MacBook Air / MacBook Pro-তে কাজ করবে?

হ্যাঁ। লেখো সব Apple Silicon ম্যাকে নেটিভভাবে চলে — MacBook Air, MacBook Pro, iMac, Mac Mini, Mac Studio, Mac Pro — যেকোনো M-সিরিজ চিপে (M1, M2, M3, M4, M5 ও ভবিষ্যতের চিপ)।

macOS-এর বিল্ট-ইন বাংলা কিবোর্ডের সাথে এটার তফাত কী?

macOS-এর বিল্ট-ইন বাংলা ইনপুট অ্যাপলের নিজস্ব লেআউট — অভ্র ফোনেটিকের নিয়মে চলে না। Windows-এ Avro Keyboard বা Linux-এ OpenBangla দিয়ে যারা অভ্যস্ত — লেখো তাদের কাছে চেনা লাগবে। একই নিয়ম, সাথে দেড় লাখ শব্দের অভিধান ও সাজেশন।

লেখো কি ফ্রি? ওপেন সোর্স?

হ্যাঁ, পুরোপুরি ফ্রি — বিজ্ঞাপন নেই, সাবস্ক্রিপশন নেই, সাইন-আপ নেই। ওপেন সোর্স, MPL-2.0 লাইসেন্সে। ভেতরের ইঞ্জিন হলো OpenBangla-র riti — Linux-এর OpenBangla Keyboard যেটা দিয়ে চলে।

২০২৬ সালে ম্যাকের জন্য সেরা অভ্র কিবোর্ড কোনটি?

Apple Silicon-এ নেটিভভাবে চলে এমন একমাত্র অভ্র ফোনেটিক কিবোর্ড এখন লেখো। iAvro এখনো Rosetta দিয়ে চলে, কিন্তু Apple ঘোষণা করেছে Rosetta ধীরে ধীরে বন্ধ হবে — macOS 27 পর্যন্ত পূর্ণ সাপোর্ট, macOS 28 থেকে শুধু পুরোনো গেমসে। লেখো ফ্রি, ওপেন সোর্স, M1–M5 ম্যাকে সরাসরি চলে।

Apple Silicon ম্যাকের জন্য iAvro-র কোনো বিকল্প আছে?

হ্যাঁ — লেখো হলো iAvro-র আধুনিক বিকল্প। Apple Silicon-এর জন্য নেটিভ, একই অভ্র ফোনেটিক নিয়ম, সাথে দেড় লাখ শব্দের অভিধান ও সাজেশন। Chrome, Safari, Notes, Mail — সব অ্যাপে চলে।

ম্যাকে কীভাবে বাংলা টাইপ করব?

লেখো ইনস্টল করুন, তারপর System Settings → Keyboard → Input Sources-এ গিয়ে + চেপে "Lekho" খুঁজে যোগ করুন। Globe কী বা Ctrl+Space দিয়ে সুইচ করুন। ইংরেজিতে লিখুন, বাংলা পাবেন — যেমন "ami" লিখলে "আমি"।

Homebrew দিয়ে লেখো ইনস্টল করা যায়?

হ্যাঁ। brew install --cask arahim3/lekho/lekho চালান — এক কমান্ডেই ইনস্টল হয়ে যাবে। আপডেট: brew upgrade --cask lekho; মুছতে: brew uninstall --cask lekho। Homebrew ছাড়াও GitHub Releases থেকে DMG নামিয়ে ইনস্টল করা যায়।
